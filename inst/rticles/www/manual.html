Crea tu primer documento

Rticles te ayudará a crear tu primer documento usando R + markdown = Rmarkdown (.Rmd).

  1. Elige que tipo de documento deseas crear.

    • Markdown: artículos, tesis y reportes.
    • Bookdown: manuales y libros.

  2. Para usar las plantillas debes tener instalado R y RStudio.

  3. Elige el nombre para tu documento.

  4. Proyectos de R (R project) te permitirá organizar mejor tus trabajos (recomendado).

  5. Seleccionada las opciones, descarga (“Download”) o crea (“Create”) las dependencias.

  6. Accede al folder donde descargaste/creaste los documentos.

  7. Click el archivo con extensión .Rproj. Se abrirá Rstudio.

  8. Abre el archivo con extensión Rmarkdown (.Rmd).

  9. Compilar el documento usando el botón Knit y esperar hasta la conclusión del proceso (se abrirá un documento en word).

  10. El documento es un ejemplo completo con todas las partes de documento técnico: tablas, figuras, referencias, formato, etc.

  11. Con el ejemplo puedes crear tu propio documento.

  12. A escribir los manuscritos! :)
